### 涩图/二次元图片
这一块分为两种调用方法

#### Lolicon API (lolicon)
顾名思义，这部分调用了Lolicon API

调用方法：在群中/私信 Hana "我要一份色图" 即可

相关API：**https://api.lolicon.app**

**请注意 为了防止服务器爆炸 这一部分的调用是 三分钟内15次**

#### 本地调用图片(nativesetu)
这一块相对来说比较简单 而且也很方便调用

速度很快~  ~~可以防止我服务器爆炸~~

方法如下: 在群中发送 "来份涩图/来份二次元/来份车万/来份猫猫"

PS：如果调用指令, Hana 有一定概率会骂你 a.a
